# Template Bun — Plan Index

> Updated: 2026-03-04

## Usage

Each plan is a single-line link to its detail file `docs/plan/PLAN-NNN.md`.

### Format

- [ ] [**PLAN-001 Short plan title**](PLAN-001.md) `YYYY-MM-DD`

### Status Markers

| Marker | Meaning |
|--------|---------|
| `[ ]` | Draft / Pending approval |
| `[-]` | Approved / In progress |
| `[x]` | Completed |
| `[~]` | Rejected / Abandoned |

### Rules

- Only update checkbox markers; never delete plan lines.
- Append new plans to the end of the list.
- See individual `PLAN-NNN.md` files for details.
