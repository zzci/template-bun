# Template Bun — Task List

> Updated: 2026-03-04

## Usage

Each task is a single-line link to its detail file `docs/task/PREFIX-NNN.md`.

### Format

- [ ] [**PREFIX-001 Short imperative title**](PREFIX-001.md) `P1`

### Status Markers

| Marker | Meaning |
|--------|---------|
| `[ ]` | To do |
| `[-]` | In progress |
| `[x]` | Completed |
| `[~]` | Closed / Won't do |

### Priority: P0 (blocker) > P1 (high) > P2 (medium) > P3 (low)

### Rules

- Only update checkbox markers; never delete task lines.
- Append new tasks to the end of the list.
- See individual `PREFIX-NNN.md` files for details.
